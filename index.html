NICTA SmartGridToolbox

Microgrid and Smart Grid Simulation

Introduction

NICTA SmartGridToolbox is a C++-11 library API for developing smart grid and microgrid simulations. It was developed by the Energy Systems subgroup of Nicta's Optimisation Research Group, with the help of the Australian National University and Actew-AGL.

SmartGridToolbox's simulation engine uses agent-based and discrete event simulation concepts. Research into future electricity grids requires the ability to handle a wide range of control strategies and simulation components. SmartGridToolbox aims to provide this ability through an API that emphasises flexibility and extensibility.

Getting started

Download or checkout a copy of SmartGridToolbox from GitHub. To download a tarball containing the latest release, go to https://github.inside.nicta.com.au/dgordon/SmartGridToolbox/releases. Unpack the archive, and cd into the top level directory. Follow the instructions in the INSTALL file.

Alternatively, to checkout a local clone of the github repository, you can 

git clone https://github.it.nicta.com.au/dgordon/SmartGridToolbox.git
(you will need access to github.it.nicta.com.au). In this case, you will need to run autogen.sh as an extra step before building, and will need the gnu autotools to be present on your system.

After you have built and installed SmartGridToolbox, you can begin to familiarise yourself with its use. At present, tutorials are still being developed. The best place to start is in the examples/MicrogridDemo folder, which sets up a simulation involving a microgrid with solar panels, a battery, a building + HVAC, and a custom controller. The source code, or doxygen documentation, are your friend.
