Detailed Description

A base class for SimComponents.

Although most SimComponents will derive from this base class, some may not, due to multiple inheritance. For example, we have SimBus : public SimComponentAdaptor, public Bus, which takes it's ComponentInterface from Bus rather than Component. Depending on needs, this kind of thing may be done using multiple inheritance, as just described, or using composition: both have advantages and disadvantages.
