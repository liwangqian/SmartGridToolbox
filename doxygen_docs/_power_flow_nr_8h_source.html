9 // Terminology:
10 // "Bus" and "Branch" refer to n-phase objects i.e. they can contain several phases.
11 // "Node" and "Link" refer to individual bus conductors and single phase lines.
12 // A three phase network involving busses and branches can always be decomposed into a single phase network
13 // involving nodes and links. Thus use of busses and branches is simply a convenience that lumps together nodes and
14 // links.
