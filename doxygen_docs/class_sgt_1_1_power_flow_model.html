Vectors.

SL node: V is a setpoint, S is a warm start. PQ node: V is a warm start, S is a setpoint. PV node: |V| is a setpoint, arg(V) is a warm start, P is a setpoint, Q is a warm start. Note that P includes both load and generation and thus may include constant power components of ZIPs. For example, for a slack bus with a constant S load, P represents the generated power minus the load draw.
